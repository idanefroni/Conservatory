2.2.1 Minumum block length

The MinBlockLength filter discards blocks with less sites than a given threshold.

Synopsis:

maf.filter=                                 \
    [...],
    MinBlockLength(min_length=1000),        \
    [...]

Arguments:

min_length={int>0}

The minimum length.
