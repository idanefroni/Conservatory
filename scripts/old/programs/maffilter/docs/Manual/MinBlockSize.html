2.2.2 Minumum block size

The MinBlockSize filter discards blocks with less sequences than a given threshold.

Synopsis:

maf.filter=                                 \
    [...],
    MinBlockSize(min_size=3),               \
    [...]

Arguments:

min_size={int>0}

The minimum size.
